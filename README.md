# homebrew-pmon

Homebrew tap for [PMON](https://github.com/So-Yul-e/PMON) — Process Monitor for macOS.

## Install

```bash
brew tap So-Yul-e/pmon
brew install --cask pmon
```

## Update

```bash
brew upgrade --cask pmon
```

## Uninstall

```bash
brew uninstall --cask pmon
brew untap So-Yul-e/pmon
```
